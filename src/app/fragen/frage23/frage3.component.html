Frage23: gebe Beispiele für die Typumwandlung von string nach number und von number nach string

Deine Lösung: 



string nach number:
  let s:string ="2";
  let n: number = +s;

  number nach string:
  let n:number=2;
  let s:string=""+n;

  number nach string mit Template string:
  let n:number=2;
  let s:string=`${{n}`;
