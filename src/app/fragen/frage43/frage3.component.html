Frage43: erkläre, was eine closure ist und gib beispiele und zeichne dann das closure ablauf diagramm

Deine Lösung: 


  closure= arrow function (javascript) & lambda expression (java)
  closure= anonymer codeblock, kann wie function aufgerufen werden, mit parametern und ist ein object, kann also in variablen gespeichert werden.
  zweck: vermeidung anonymer innerer klassen mit nur 1 methode
  syntax: () => {{'{'}} }  //bevorzugt
  oder: function // nicht benutzen

  Beispiele:
  dynamisch: (param1, param2) => {{'{'}} let ergebnis = …; return ergebnis; }
  mit nur 1 param: param => {{'{'}} let ergebnis = …; return ergebnis; }

  statisch: (param1: number, param2: number): number =>
  {{'{'}} let ergebnis: number = …; return ergebnis; }

  statischer closure typ (als variable):
  const funcvar: (n: number) => number = (n: number): number => {{'{'}} return n+5; }

  diagramm: deklaierer->closure->aufrufer
  1. deklaierer erstellt closure
  2. methode mit closure als parameter aufrufen
  2.1 closure ausführen
