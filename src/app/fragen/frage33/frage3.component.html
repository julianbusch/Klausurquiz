KAPITEL TYPESCRIPT:


Frage33: nenne die Unterschiede von Typescript Klassen und Java Klassen

Deine Lösung: 



construktor: ist wie in java, allerdings heißt er immer constructor().
  vererbung: wie in Java ist keine mehrfachvererbung möglich, allerdings müssen konstruktoren von unterklassen immer super() aufrufen
interface-implementierung: ist wie in java
  public/protected/private,public ist default, nicht protected wie in java
  protected bezieht sich ausschließlich auf Vererbung, beinhaltet im Gegensatz zu Java nicht die Klassen des selben Moduls sonst wie in Java
  statische Attribute und Methoden: Schlüsselwort static, wie in Java
abstrakte Klassen und Methoden: Schlüsselwort abstract, wie in Java
