KAPITEL TYPESCRIPT:


Frage25: gebe ein Beispiel für einen typecast

Deine Lösung: 



let specht:Dozent=new Professor("Thomas", "Specht", "SPE");
  let raum:string=specht.raum; // compilefehler, klasse Dozent hat kein attribut raum, daher typecast notwendig
  let raum2:string=(specht as Professor).raum; //typecast mit as
  let raum3:string=(<>Professor>specht).raum; // funktioniert auch
  let raum4:string=(Professor)specht.raum; //funktioniert NICHT in typescript, nur in zb java
