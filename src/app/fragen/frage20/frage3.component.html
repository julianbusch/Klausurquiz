KAPITEL TYPESCRIPT:


Frage20: erkläre, was ein assoziatives Array ist und gib ein Beispiel

Deine Lösung: 



besteht aus Key Value paaren, Key ist vom Typ string, key braucht kein "" / '',
  value kann beliebigen typ haben, sieht aus wie JSON

  dynamisch typisiert:
  const autokennzeichen = {{
  KA: "Karlsruhe",
  S: "Stuttgart
  };

  statisch typisiert:
  const autokennzeichen : {[key:string]:string}= {{
  KA: "Karlsruhe",
  S: "Stuttgart
  };

  Elementzugriff mit Array:
  let stadt= autokennzeichen["KA"];

  Elementzugriff als objekt:
  autokennzeichen.KA;
