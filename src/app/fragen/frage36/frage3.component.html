Frage30: gebe ein Beispiel für Getter und Setter

Deine Lösung: 



class Professor{{'{'}}
  private _raum: string; // _ muss vor die variable, damit kein namenskonflikt mit der get methode raum() entsteht

  constructor(public vorname:string, public nachname:string, raum?:string){{'{'}}
  this._raum=raum;
  }

  get raum():string{{'{'}}
  return this._raum; / return `Raum ${{'{'}} this._raum}`;  // rechts mit template string
  }

  set raum(neuerRaum: string){{'{'}}
  this._raum=neuerRaum;
  }
  }

  verwendung der getter und setter:
  let specht:Professor=new Professor("Thomas", "Specht", "109");
  specht.raum="10000"; // dies ruft den setter auf und ändert den wert
  document.body.innerHTML=specht.raum; // dies ruft den getter auf, also kommt in das DOM der neue Wert "10000"
