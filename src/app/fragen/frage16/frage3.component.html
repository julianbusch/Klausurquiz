Frage16: erkläre die Unterschiede zwischen den Datentypen object und any

Deine Lösung: 



any kann einen beliebigen Typ aufnehmen, object nur nicht elementare Datentypen
  any ist gleichwertig, wie garkein Datentyp angegeben,
  any hat Zugriff auf Attribute und Methoden abhängig des Typs,
  aufruf von nicht vorhandenen Methoden und Attributen führt zum Laufzeitfehler

  object lässt nur aufrufe von Methoden und Attributen der Klasse object zu unabhängig vom objecttyp
