Frage24: gebe Beispiele für die typkonvertierung von string nach boolean und boolean nach string

Deine Lösung: 



string nach boolean (durch vergleich mit true):
  let s:string="false";
  let b:boolean=s=="true";

  string nach boolean (durch vergleich mit true und false):
  let s:string = "false";
  let b:boolean=s=="true"? true:s=="false"? false:undefined;

  boolean nach string:
  let b:boolean:"false";
  let s:string=""+b;

  boolean nach string mit template string:
  let b:boolean="false";
  let s:string=`${{b}`;
