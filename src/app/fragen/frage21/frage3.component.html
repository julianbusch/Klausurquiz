Frage21: gebe ein Beispiel für die deklaration und aufruf einer enum

Deine Lösung: 



enum Color{{
  Red,
  Green,
  Blue
  }     // kein ; am ende, wie bei klassen

  deklaration von enum variablen:
  let c: Color= Color.Red; //statisch
  let c2=Color.Green;  //dynamisch

  Zugriff:
  Color[c]; // == Red

  Aufzählungen von enums:
  enum Monat{{

  }
