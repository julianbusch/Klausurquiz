KAPITEL TYPESCRIPT:


Frage39: erkläre, was ein typescript modul ist

Deine Lösung: 



typescript datei wird zum modul, wenn auf oberster ebene ein expor/import befehl ist. typescript modul = eine typscript datei, mit eigenem scope. typescript module gibt es erst seit ECMAscript 2015 (es2015).
  wenn man die javascript datei als modul verwenden will muss man dat attribut {{'type="module"'}} beim einbinden der js datei in die index html schreiben.
  außerhalb eines moduls ist nur sichtbar, was mit export gekennzeichnet ist, hierzu gibt es 2 möglichkeiten:

  direkt an der klasse:
  z.B.: export class Dozent {{'{'}} /* ... */ }

  als sammlung:
  export {{'{'}} ding1, ding2, ding3 };
  z.B.: export {{'{'}}Dozent, Professor, Lehrbeauftragter};

  wenn eine typescript datei eine klasse importiert wird sie auch zum modul. Import immer als .js datei, nicht .ts. bsp:
  import {{'{'}} Dozent, Professor, Lehrbeauftragter } from './dozent.js'

  wenn man alle exporte aus einem modul importieren will, benutzt man
  import * as aliasname from "";

  z.B.: import * as dozenten from './dozent.js';

  aufruf: mit aliasname.Klassenname
   Beispiel: let gerten: dozenten.Professor = new dozenten.Professor( /* ... */ );
