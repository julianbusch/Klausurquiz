Frage49: erkläre, was inperpolationsbindung ist und gib ein Beispiel

Deine Lösung: 



Zugriff in der View auf Attribute der ModelView, aufruf durch {{'{{attributname (2 geschweifte klammern zu)'}}


  beispiel:
  int typescript datei (ModelView):
  export class AppComponent {{'{'}}
  attributname = 'Attributwert';
}

  dazugehörige view (html template):
  "<"h1">"{{ '{{attributname (2 geschweifte klammern zu)'}}"<"h1">"
