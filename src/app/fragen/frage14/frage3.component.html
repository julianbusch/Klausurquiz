Frage14: Was ist ein Template String?

Deine Lösung: 


  mit Template String kann man Berechnete Ausdrücke, sowie den Wert von Variablen
  in einem String verwenden
  Backquote ` auf Tastatur: Shift-Accent Space
  kann über mehrere Zeilen gehen kann berechnete Ausdrücke (Closures) enthalten
